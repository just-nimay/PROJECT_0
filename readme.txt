РУССКИЙ
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=
	Для запуска игры:
	скачать архив с гитхаба: https://github.com/just-nimay/PROJECT_0.git (нажмите кнопку "code" -> нажмите download ZIP)
	установить python версии не менее 3.6
	откройте консоль и введите (это уснановка библиотек для Python)
		ДЛЯ LINUX
			sudo apt install python3-pip
			cd "путь к скачанной папке PROJECT_0"
			python3 start.py
		ДЛЯ WINDOWS
			cd "путь к скачаной разархивированной папке PROJECT_0" (например: cd "C:/users/ivan/downloads/PROJECT_0")
			python3 start.py

ENGLISH
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=
	To start the game:
	download the archive from github: https://github.com/just-nimay/PROJECT_0.git (click "code" -> click download ZIP)
	install python version at least 3.6
	open the console and type (this is the installation of libraries for Python)
		FOR LINUX
			sudo apt install python3-pip
			cd 'foldeer path PROJECT_0' (for example: cd 'C:/users/ivan/downloads/PROJECT_0/' (PROJECT_0 -- this is in the archive of this repository))
			python3 start.py
		FOR WINDOWS
			cd "path to the downloaded unzipped PROJECT_0 folder" (for example: cd "C:/users/ivan/downloads/PROJECT_0 ")
			python3 start.py
