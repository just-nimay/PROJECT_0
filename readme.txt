РУССКИЙ
	Для запуска игры:
	скачать архив с гитхаба: https://github.com/just-nimay/PROJECT_0.git (нажмите кнопку "code" -> нажмите download ZIP)
	установить python версии не менее 3.6
	откройте консоль и введите (это уснановка библиотек для Python)
		ДЛЯ LINUX
			sudo apt install python3-pip
			python3 -m pip install kivy[base]
			pip3 install playsound (установка библиотеки Playsound)
			pip3 install matplotlib (установка библиотеки matplotlib)
			cd 'путь к файлу __main__.py' (например: C:/users/ivan/downloads/PROJECT_0/content/__main__.py (PROJECT_0/content/__main__.py -- это находится в архиве этого репозитория))
			python3 __main__.py (запуск игры)
		ДЛЯ WINDOWS
			cd "путь к скачаной разархивированной папке content" (например: cd "C:/users/ivan/downloads/PROJECT_0/content")
			python3 get-pip.py
			python3 -m pip install
			python3 -m pip install kivy[base]
			pip3 install playsound (установка библиотеки Playsound)
			pip3 install matplotlib (установка библиотеки matplotlib)
			python3 __main__.py (запуск игры)
ENGLISH
	To start the game:
	download the archive from github: https://github.com/just-nimay/PROJECT_0.git (click "code" -> click download ZIP)
	install python version at least 3.6
	open the console and type (this is the installation of libraries for Python)
		FOR LINUX
			sudo apt install python3-pip
			python3 -m pip install kivy[base]
			pip3 install playsound (installing the Playsound library)
			pip3 install matplotlib (installing the matplotlib library)
			cd 'file path __main__.py ' (for example: C:/users/ivan/downloads/PROJECT_0/content/__main__.py (PROJECT_0/content/__main__.py -- this is in the archive of this repository))
			python3 __main__.py (starting the game)
		FOR WINDOWS
			cd "path to the downloaded unzipped content folder" (for example: cd "C:/users/ivan/downloads/PROJECT_0/content ")
			python3 get-pip.py
			python3 -m pip install kivy[base]
			pip3 install playsound (installing the Playsound library)
			pip3 install matplotlib (installing the matplotlib library)
			python3 __main__.py (starting the game)
